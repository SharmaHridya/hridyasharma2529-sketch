### 🤖 AI Recognition Music App
> A webcam-powered music app that uses AI body/wrist detection to trigger songs. Hold your **left wrist** in front of the camera to play the Peter Pan theme, or your **right wrist** for the Harry Potter theme — no keyboard needed!

- **Tech Stack:** HTML · CSS · JavaScript · AI/ML (body recognition)
- **Highlights:** Real-time wrist detection via webcam · Gesture-controlled audio playback
- 🔗 [Live Demo](https://hridyasharma2529-sketch.github.io/AI-Recognition-Music-App/) · [View on GitHub](https://github.com/hridyasharma2529-sketch/AI-Recognition-Music-App)

---

### 🎵 Drum Kit
> An interactive browser-based drum set where you can play drum sounds by pressing keyboard keys or clicking on-screen drum pads. Rock out without leaving your browser!

- **Tech Stack:** HTML · CSS · JavaScript
- **Highlights:** Keyboard & click controls · Real drum sound samples · Animated key press feedback
- 🔗 [View on GitHub](https://github.com/hridyasharma2529-sketch)

---

### 🎲 Dice Roll Game
> A two-player dice rolling game where each player rolls a die and the higher number wins. Refresh to roll again — simple, fast, and fun!

- **Tech Stack:** HTML · CSS · JavaScript
- **Highlights:** Random dice generation · Dynamic image updates · Instant winner display
- 🔗 [View on GitHub](https://github.com/hridyasharma2529-sketch)

---

### 🧠 Simon Game
> A browser recreation of the classic Simon memory game. The game flashes a sequence of colours and you must repeat it back — with each round adding one more step. How far can you go?

- **Tech Stack:** HTML · CSS · JavaScript
- **Highlights:** Progressive difficulty · Sound + visual feedback · Strict mode tracking
- 🔗 [View on GitHub](https://github.com/hridyasharma2529-sketch)
